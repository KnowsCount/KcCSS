## KcCSS 是甚？

一个为 [Typora](https://typora.io) 设计的美丽主题

- 现代化、极简的 CSS 代码。
- 为文字编辑排版而生。
- 和 Typora 一样极简。
- 无缝，因而无干扰。
- ...

**注意：项目目前还在初始阶段，不具备可达到的所有功能。**

## Usage

1. clone 这个仓库:
```bash
git clone https://github.com/KnowsCount/KcCSS.git
```
2. 把 `kc.css` 放到你的 Typora 主题库里
3. 选用主题
